# Glossary

A reference for common terms and acronyms in VennWL-iOS.

## General
- **White-label:** A product rebranded for multiple clients
- **Brand:** A specific client or app variant
- **Module:** A reusable UI component defined by the server
- **MVVM:** Model-View-ViewModel architecture

## Automation
- **CI/CD:** Continuous Integration/Continuous Deployment
- **Fastlane:** Automation tool for building and deploying iOS apps
- **CircleCI:** Cloud-based CI/CD platform

## Code
- **.xcconfig:** Xcode configuration file
- **Entitlements:** Permissions and capabilities for an app
- **Keychain:** Secure storage for sensitive data
