# Key Classes & Files

A quick reference to important classes and files in VennWL-iOS.

## Modules
- **VLayoutItem:** Represents a dynamic module from server JSON ([docs](../modules/overview))
- **VTextBoxCell:** Renders a text box module
- **ModularHomeViewController:** Renders homepages from modules

## Services
- **ProductService:** Fetches product data ([docs](services-networking))
- **AuthenticationService:** Handles login and token management

## View Models
- **ProductViewModel:** Transforms product data for the UI
- **ModularHomeViewModel:** Manages homepage modules

## Utilities
- **ConfigManager:** Loads and manages app configuration
- **ThemeManager:** Handles theming and appearance

## Automation
- **Fastlane/Fastfile:** Build and deployment automation ([docs](../branding/fastlane-circleci))
- **.circleci/config.yml:** CI/CD workflows
