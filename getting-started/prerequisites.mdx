# Prerequisites

Before you begin working with VennWL-iOS, ensure you have the following tools and accounts set up:

- **Xcode** (version 16.0.0 or later)
- **Homebrew** (for installing dependencies)
- **Ruby** and **Bundler** (for Fastlane and automation)
- **Tuist** (for project generation)
- **Fastlane** (for build and deployment automation)
- **Git** (for version control)
- Access to the private repositories and credentials (if required)

> **Tip:** See [Setup](setup) for installation instructions.
