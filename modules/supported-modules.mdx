# Supported Modules

Below is a list of modules supported by VennWL-iOS. Each module can be configured via server JSON and rendered dynamically.

## Content Modules
- **VTextBox:** Displays styled text. Example: banners, section headers.
- **VGallery:** Image gallery or carousel.
- **VExpandingTextView:** Expandable/collapsible text content.

## Product Modules
- **VProductInfoBar:** Shows product details (price, name, etc.).
- **VProductGrid:** Grid of products, often for recommendations or categories.
- **VProductPhotoGrid:** Grid of product images.
- **VProductRecommendations:** Personalized product suggestions.

## Layout & Utility Modules
- **VSpacer:** Adds vertical space between modules.
- **VHorizontalStackButton:** Row of buttons for actions or navigation.
- **VFixedWidthButton:** Button with a fixed width, often for CTAs.

## Other Modules
- **VCategoryRow:** Displays a row of categories.
- **VImageWithPadding:** Image with custom padding.
- **VCountdown:** Countdown timer for sales/events.
- **VWebView:** Embeds a web page.

> **Tip:** See the code in `VLayoutItemModuleType` for the full, up-to-date list.
